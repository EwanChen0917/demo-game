# 游戏设计大纲

中国古代世家题材模拟经营 + 策略游戏。玩家以一介白身起家，通过科举入仕、从军建功、经营产业、联姻结盟，将家族发展成一门三宰相、四世两封王的顶级世家。

---

## 核心循环

```
培养族人属性
    ↓
科举 / 从军 → 获得官职 / 爵位
    ↓
官职带来俸禄 + 声望 + 人脉
    ↓
声望吸引门客 / 促成联姻
    ↓
下一代继承，属性更强
    ↓
家族势力扩大，参与朝堂争斗
    ↓
（终局）权倾朝野 / 另立新君
```

---

## 时间系统

- 基本单位：**季**（春夏秋冬）
- 每季结算：俸禄收入、资产收益、NPC 关系变化
- 每年结算：族人年龄增长、科举冷却递减、声望自然衰减
- 玩家手动触发「过一季」推进时间

## 族人属性设计

属性分两层：**数值属性**（连续值 1-100，可培养）+ **标签特质**（离散，有或无，稀有）。

---

### 数值属性

所有族人出生时都有全套数值属性，区别只在高低。

#### 当前已实现

| 属性 | 说明 | 驱动系统 |
|---|---|---|
| 文才 | 读书著述天赋 | 科举通过率 |
| 武力 | 武艺兵法天赋 | 军功晋升速度、战斗胜率 |
| 运气 | 综合运势 | 科举加成、随机事件结果偏向 |

#### 预留数值属性（随对应系统上线时实装）

| 属性 | 解锁时机 | 驱动系统 | 设计说明 |
|---|---|---|---|
| **魅力** | 世家关系系统 | 外交好感提升速度、联姻成功率、门客招募 | 高魅力族人每季自动增加与周边世家的好感度；担任外交职位（如鸿胪寺）时效果翻倍 |
| **谋略** | 朝堂争斗系统 | 计谋成功率、识破阴谋、军事部署加成 | 高谋略族人可主动发起"构陷"行动；被构陷时有概率识破；带兵时提升战术加成 |
| **辩才** | 朝堂争斗系统 | 朝堂辩论、游说中立势力、谈判 | 弹劾/被弹劾事件中影响胜负；游说其他世家站队时提升成功率 |
| **商道** | 资产经营系统 | 产业收益倍率、低买高卖差价、商号利润 | 高商道族人管理商铺时收益 +N%；可触发"商机"随机事件获得额外财富 |
| **医** | 寿命/战斗系统 | 族人寿命延长、战场救治、医官职位 | 高医族人可为他人续命（+3~8年寿命上限）；战斗中己方伤亡概率降低；可担任太医院职位 |

---

### 标签特质

特质是**稀有能力**，不是数值，只有有/无两种状态。分为**先天特质**（出生随机获得）和**后天特质**（特定事件触发解锁）。

同一族人最多同时持有 **4 个特质**（先天 + 后天合计）。

#### 先天特质（出生时随机概率触发）

| 特质 | 稀有度 | 效果 |
|---|---|---|
| **巫灵** | 罕见（5%） | 可主持祭祀仪式，触发庇护事件（当季全家运气 +15）；巫值高时可对敌对世家施加诅咒 |
| **卜算** | 罕见（5%） | 每季可占卜预览下季随机事件吉凶；参与重大决策时成功率有隐性 +10% |
| **异象** | 极罕见（1%） | 出生时触发全局异象事件，家族声望暴涨；成年后随机解锁一条隐藏剧情线 |
| **将星** | 稀有（8%） | 带兵时士气加成 +20%；军功晋升所需战斗次数减半 |
| **文曲** | 稀有（8%） | 科举所有阶段通过率额外 +15%；有机会直接跳级（如童生直接考举人） |
| **商贾之才** | 普通（15%） | 所有产业收益 +10%；可识别"黑市"随机事件并从中获益 |
| **长寿之相** | 普通（12%） | 寿命上限 +15 年 |
| **短命之相** | 普通（10%） | 寿命上限 -10 年（负面特质，增加游戏难度） |
| **体弱多病** | 普通（10%） | 每年有概率触发患病事件，需消耗医药费 |
| **克父** / **克母** | 普通（6%） | 出生时父或母有概率触发意外死亡事件（高风险高收益：该族人武力/文才随机 +20） |

#### 后天特质（特定条件下解锁）

| 特质 | 解锁条件 | 效果 |
|---|---|---|
| **状元及第** | 科举考中状元 | 文才上限突破至 120；担任文官时声望加成翻倍 |
| **百战老将** | 累计参战 10 次以上 | 武力上限突破至 120；带兵时有概率触发"以少胜多"事件 |
| **富甲一方** | 家族财富超过 100000 金 | 商道影响的所有收益额外 +20%；可触发"皇商"剧情线 |
| **朝中有人** | 同时有 3 名以上族人担任四品以上官职 | 科举打点加成 +10%；弹劾事件中天然占优 |
| **声名狼藉** | 被弹劾成功 3 次以上 | 科举通过率 -10%；但部分"黑道"随机事件收益翻倍（负面特质） |
| **方外之人** | 主动退出仕途（辞官）后触发 | 不能再入仕；但每季为家族带来固定声望 +5，且触发的随机事件以吉祥类为主 |
| **结义兄弟** | 与另一世家族人发生"结义"随机事件 | 与该族人所在世家好感度永久 +30；双方家族发生冲突时该族人不参与 |

---

### 数值属性通用规则

```
出生值 = (父属性 + 母属性) / 2 ± 随机波动(-10 ~ +10)
运气：纯随机(20~80)，不受父母影响
医、巫灵相关：父母均有时，子代获得对应特质的概率翻倍
```

数值上限通常为 100；持有「状元及第」「百战老将」等特质后可突破上限。
数值下限 1，出生时不低于 10。

## 科举系统设计

### 阶段与对应官职

| 阶段 | 难度 | 授予官职 | 冷却 |
|---|---|---|---|
| 童生 | 易（80% 基础） | 无 | 1 年 |
| 秀才 | 中（60% 基础） | 无 | 1 年 |
| 举人 | 难（40% 基础） | 无 | 3 年 |
| 进士 | 很难（25% 基础） | 正七品知县 | 3 年 |
| 状元 | 极难（10% 基础） | 正五品知府 | 3 年 |

### 通过率公式

```
通过率 = 基础率 + (文才 - 50) × 0.4 + (运气 - 50) × 0.2
通过率上限 95%，下限 5%
```

## 官职体系

文武各九品十八级，品级越小越高（正一品最高）。

- **文官路线**：科举入仕，逐级升迁至宰相
- **武将路线**：从军建功，封侯拜爵，获得封地

俸禄随品级递增，从九品 8 金/季到一品 200 金/季。
